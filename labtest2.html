Introduction to C++ Programming

Course Curriculum

Course Overview

This course introduces the fundamentals of C++ programming, from basic syntax and data types to object-oriented programming, STL, and file handling. By the end of this course, students will be able to design, implement, and debug C++ programs.

Duration

10–12 Weeks

Prerequisites

Basic knowledge of computers, logic, and mathematics

Level

Beginner to Intermediate

Tools

Any C++ Compiler (GCC, MinGW, Turbo C++, etc.) & IDE

Module 1: Introduction & Basics of C++
Week 1
  • History and features of C++
  • Structure of a C++ program
  • Compiling and running a C++ program
  • Header files, main() function, comments
  • Input and output (cin, cout)
Module 2: Data Types, Operators & Control Statements
Week 2–3
  • Variables and constants
  • Data types (int, float, char, bool, etc.)
  • Type casting
  • Arithmetic, relational, logical, and bitwise operators
  • Conditional statements (if, if-else, switch)
  • Loops (for, while, do-while)
  • Break, continue, and goto (theory)
Module 3: Functions & Arrays
Week 4–5
  • Function declaration and definition
  • Call by value and call by reference
  • Inline functions and default arguments
  • 1D and 2D arrays
  • String handling in C++ (C-style and std::string)
Module 4: Pointers & Dynamic Memory
Week 6
  • Pointer basics and pointer arithmetic
  • Pointers and arrays
  • Pointers and functions
  • Dynamic memory allocation (new, delete)
Module 5: Object-Oriented Programming in C++
Week 7–8
  • Classes and objects
  • Access specifiers (public, private, protected)
  • Constructors and destructors
  • Inheritance (single, multiple, multilevel)
  • Polymorphism (function overloading, operator overloading)
  • Virtual functions and introduction to runtime polymorphism
Module 6: Templates, Exception Handling & STL
Week 9–10
  • Function templates and class templates
  • Exception handling (try, catch, throw)
  • Introduction to Standard Template Library (STL)
  • Vectors, lists, stacks, queues, maps (basic usage)
Module 7: File Handling & Project
Week 11–12
  • File streams (ifstream, ofstream, fstream)
  • Reading from and writing to files
  • Binary file operations (basic)
  • Mini Project in C++ combining all concepts
  • Revision and practice problems
